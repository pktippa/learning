# Java Data Types

Java is a strong statically typed language.

Java has two categories of data
* Primitive Type 
* Object Type

## Primitive Types

* Boolean - true / false - 1 bit
* Char - Unicode character - 16 bits
* Byte - 8 bits - -128 to 127 (2**8 = 256)
* Short - 16 bits
* Int - 32 bits
* Long - 64 bits - 100L
* Float - 32 bits
* Double - 64 bits

And

* String
